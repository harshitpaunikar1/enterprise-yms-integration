# Enterprise YMS Integration for Logistics

> **Domain:** Logistics

## Overview

Scalable integration backbone connecting yard management to ERP, WMS, and recognition systems through REST/JSON APIs, SFTP, and event streaming with OAuth2 security.

## Skills & Technologies

- Enterprise System Integration
- REST APIs & Webhooks
- Data Mapping & ETL
- Event-Driven Architecture
- OAuth2/SSO
